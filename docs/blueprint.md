# **App Name**: AJDraw

## Core Features:

- Create Room: Generate a unique 6-digit room code for users to join and start drawing together.
- Join Room: Allow users to enter a 6-digit room code to join an existing drawing session.
- Real-time Drawing Sync: Synchronize drawing actions between all connected clients in the same room using WebSockets for a collaborative experience.
- Color and Stroke Selection: Enable users to choose colors and adjust stroke widths for their drawings.
- Canvas Clearing: Provide a clear canvas function that instantly clears the canvas for all users in the room.
- Undo Functionality: Implement an undo feature to revert the last drawing action.
